# PWA Screenshots

This directory contains screenshots for the SYNCRO PWA manifest.

## Generated Screenshot ✅

- ✅ `dashboard.png` (1280x720) - Placeholder screenshot showing SYNCRO branding

## Generation

The placeholder screenshot is generated using:
```bash
npm run generate-pwa-screenshot
```

This creates an SVG-based placeholder with SYNCRO branding.

## Production Note

For production PWA deployment, replace the placeholder with an actual screenshot of your dashboard interface showing realistic content and usage.

## Guidelines

- Capture the main dashboard interface
- Use 1280x720 resolution for wide form factor
- Show the app in a realistic usage state
- Ensure good quality and readability
